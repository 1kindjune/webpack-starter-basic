I'm a a simple webpack starter project for your basic web development needs.

Star
No wizard code, understandable, scalable.
A minimal yet rich set of features to get you going.
  • Use the latest JavaScript
  • Write styles in Sass
  • Load and bundle your assets
  • Start a development server and preview your work
  • Automatic css vendor prefixes
  • Optimizes scripts for production
  • Optimizes styles for production
git clone https://github.com/lifenautjoe/webpack3-starter
cd webpack3-starter
npm run start # start the development server
npm run build # build for production
