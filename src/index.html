I'm a simple webpack3 starter project for all your web development needs.

Star
No wizard code, understandable and scalable.
A minimal setup of plugins to help you get started and separated configuration files for development and production environments.
  • babel-loader to load your scripts with ES6
  • sass-loader to load your styles
  • url-loader to load your assets
  • webpack-dev-server to live preview your work
  • uglifyjs-webpack-plugin to optimize your scripts for production
  • optimize-css-assets-webpack-plugin to optimize your styles for production
git clone https://github.com/lifenautjoe/webpack3-starter
cd webpack3-starter
npm run start // start the development server
npm run build // build for production
